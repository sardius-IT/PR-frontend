# ImmigraPath — Next.js + Tailwind
A clean, modern immigration consultancy website scaffold (App Router) inspired by Croyez, VJC Overseas, and Visas Avenue (original content).

## Quick Start
```bash
npm install
npm run dev
# open http://localhost:3000
```

## Pages
- `/` — Hero, KPIs, programs
- `/services` — PR, Study, Work
- `/countries` — Destinations
- `/about` — Why us
- `/blog` — Articles (placeholders)
- `/contact` — Free assessment form (client-side demo)

## Customize
- Colors and components in `app/globals.css` and `components/*`
- Add images with `next/image` (Unsplash demo on home)
- Wire `/contact` form to your backend, Formspree, or Google Apps Script

## Notes
- App Router (Next 14), TailwindCSS 3 — no DB/back-end included
- All copy is original and generic — replace with your brand details
